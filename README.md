Bike Buyers Analysis - Excel Project

A complete data analysis project in Microsoft Excel that examines the relationship between lifestyle and demographic attributes and bike buying behaviour in Europe, Pacific and North America.

Project Overview

The project explores a data set of 1,000+ people to understand who and why people purchase bikes. It demonstrates the entire data analytics process - from data to dashboard - using just Microsoft Excel.

File Structure

|Sheet	   |Description|
|-------------|-------------|
|bike_buyers  |Original data set (14 columns and 1,000+ rows)|
|Pivot Tables |Pivot tables summary of bikebuyers by various demographic|
|Dashboard	   |Dashboard with key takeaways|


Dataset Columns

| Column           | Description |
|------------------|--------------|

| ID               |  Customer ID |
| Marital Status   | Married or Single |
| Gender           | Male or Female |
| Income           | Annual income (USD) |
| Children         | Number of children |
| Education        | Level of education (e.g. Bachelor's, Master's) |
| Occupation       | Type of job (e.g. Professional, Clerical, Manual) |
| Home Owner       | Home ownership status (Yes/No) |
| Cars             | Number of cars owned |
| Commute Distance | Commuting distance (e.g. 0-1 Miles, 5-10 Miles) |
| Region           | Customer's region (Europe, Pacific, North America) |
| Age              | Customer age |
| Age Group        | Age group (Teen, Middle Age, Senior) |
| Bike Purchased   | Outcome - Yes or No |


Key Insights

-Customers in their middle age are most likely to buy a bike
- High income earners are more likely to purchase a bike, especially Professionals
- Short trips (0-1 miles) are common, so bikes are likely to be used for convenience
- Singles and Homeowners have a high purchase rate

Tools Used
- Microsoft Excel- cleaning data, pivot tables, dashboard and charts
